# Grama Kalyana Sports - Kabaddi 🏃‍♂️🤼

Grama Kalyana Sports is an Android application specially designed for organizing and managing Kabaddi matches in villages and rural communities.

The app helps Kabaddi teams to:

- Book Kabaddi grounds
- Organize matches
- Post match challenges
- Track match scores
- Manage team schedules

This project promotes traditional rural sports culture through digital technology.

---

# 📌 Problem Statement

In many villages, Kabaddi grounds are used without any proper scheduling system.  
Teams often face problems such as:

- No ground booking system
- Difficulty organizing matches
- Clash between multiple teams
- No digital score tracking
- Poor communication between villages

This creates confusion and reduces organized sports participation.

---

# 🎯 Project Objective

The main objective of Grama Kalyana Sports is to create a digital platform for managing village Kabaddi activities efficiently.

The app allows users to:

- Book Kabaddi ground slots
- Organize friendly Kabaddi matches
- Post and accept match challenges
- Maintain Kabaddi score records
- Encourage youth participation in traditional sports

---

# 🚀 Features

## 📅 Ground Booking
- View available Kabaddi ground slots
- Book time slots easily
- Prevent duplicate bookings

## ⚔️ Match Challenge Board
- Post Kabaddi match challenges
- Accept or reply to challenges
- Organize inter-village Kabaddi matches

## 🧾 Kabaddi Score Board
- Upload match scores
- View recent match results

## 👥 Team Management
- Create and manage Kabaddi teams
- Store player and match details

## 🔔 Notifications
- Match reminders
- Booking confirmations

# 📱 App Workflow

1. User opens the app
2. Selects Kabaddi ground
3. Checks available slots
4. Books preferred slot
5. Posts or accepts Kabaddi challenges
6. Match scores are updated after gameplay

# 🔥 Technical Implementation

- Firebase used for real-time challenge updates
- Room Database used for storing team details and match history
- Slot conflict prevention implemented
- Responsive sports-themed UI
